## Lesson02

実際に React を起動し,必要なレイアウトを自分で作成しよう！

- [ ] ターミナルで`yarn install`と叩く
- [ ] ターミナルで`yarn start`と叩き,アプリを起動する
      （自動でブラウザが開かない場合は`http://localhost:3000/`へアクセスする）
- [ ] ブラウザで`Hello React!!`を確認する
- [ ] App.jsx ファイルの return 内にコードを書き,Mock 画像のような見た目にする
